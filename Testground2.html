Course Description & Objectives

Coding for Business adopts a cohesive set of fundamental scripting and programming skills of HTML, CSS, JavaScript, and ReactJS library essential for frontend web development. The course serves to set the stage for the four key technologies in MERN stack development:

  • MongoDB
  • Express JS
  • React JS
  • Node JS

Learning and Teaching Structure

Participants will be required to demonstrate strong problem solving and logic thinking skills during the process of tackling problems in prototyping web applications.

  10. Grade plus assignment - Grade plus will not be granted unless exceptional cases.
